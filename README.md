# ReverseEngineering
В данном репозитории хранятся мои крякнутые таски (из CTF, курсов, просто старых и интересных мне приложений, с сайтов по RE).
К каждой таске прикреплен файл с решением (объяснением). Делаю это по причине того, что полезно документировать свою работу и свой прогресс.
Соответственно, где таски простые прикреплены текстовые документы solution.cpp с объяснением, если сложные - подробные README.md.
